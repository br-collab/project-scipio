# Security Policy

## Scope

Project Scipio is an engineering prototype and learning environment. It is not intended for operational deployment or production use.

## Reporting a vulnerability

If you discover a security issue, please do not post it publicly in an issue first.

Instead, report it privately to the repository owner with:

- a description of the issue
- reproduction steps
- affected files or routes
- potential impact

## Response expectations

Because this project is a prototype, response times are best effort rather than SLA-backed.

## Operational note

Do not use this repository to store sensitive real-world operational data, credentials, or classified material.
